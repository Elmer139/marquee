# 🎬 Marquee

A personal, phone-friendly catalog of the movies, TV, and music you love — organized to fine detail, with **Signature Moments** (the scenes/clips worth sharing), fun-fact trivia, emotional collections, sorting, and always-current "where to watch / listen" links.

Built as a single self-contained `index.html` (no build step, no dependencies) so it runs anywhere and never rots.

## Features
- **Screen** (movies & TV) and **Sound** (music) worlds, each with its own accent color
- **Signature Moments** — per-item scene/clip links, framed "great even if you've never seen it"
- **Copy to share** — one tap copies title + fun fact + moment links to paste into a text
- **Collections** — emotional groupings (True Love, Sorrow, Mastery, Tear Jerkers, Spiritual)
- Cards + Table views, click-header sorting, search
- **▶ Trailer** button on every film; light/dark theme
- Add items yourself — saved to your device (localStorage)
